# Netflix Data Analysis

## Project Overview
This project analyzes Netflix movies and TV shows using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Objectives
- Analyze Netflix trends
- Compare Movies vs TV Shows
- Visualize genre distribution
- Study release patterns

## Dataset
Netflix Titles Dataset

## Key Insights
- Most Netflix content was added after 2015
- Movies dominate the platform
- Drama is one of the most common genres
